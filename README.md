# Retail Inventory Analytics

## Objective
Analyze retail inventory data using Python and Google Colab.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Tasks Performed
- Data Understanding
- Data Cleaning
- Exploratory Data Analysis
- Data Visualization
- Business Insights

## Visualizations
- Bar Chart
- Pie Chart
- Histogram
- Line Graph
- Correlation Heatmap

## Outcome
Identified best-selling products, low-stock items, supplier contributions, and inventory optimization opportunities.
